# Collage-Application

This repository contains a set of C# applications that implement a **science game** and a **math game** designed as interactive learning tools. Each game demonstrates basic game logic, user interaction, and problem-solving mechanics using C#.

## Projects Included

This repository contains the following main projects:

- **Math game** – A game focused on mathematics challenges.
- **science game** – A game focused on science-related questions.
- **test** – A test/experimental project.
- **Solution files (.sln)** for each project.

## Features

- Interactive science‑based questions
- Math challenges with varying difficulty
- Score tracking and immediate feedback
- Written entirely in C#
- Separate solutions for each game module

## Technologies Used

- C#
- .NET Framework (version may vary by project)
- Visual Studio (recommended IDE)
