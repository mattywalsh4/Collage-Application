# Science and Math Game (C#)

A C# application that provides interactive science and mathematics games designed to support learning through simple gameplay. This project was created as a learning exercise and demonstrates basic game logic, user interaction, and problem-solving mechanics using C#.

## Features

- Interactive science-based questions
- Math challenges with varying difficulty
- Score tracking and feedback
- Simple and intuitive user interface
- Written entirely in C#

## Technologies Used

- C#
- .NET Framework or .NET 
- Visual Studio
